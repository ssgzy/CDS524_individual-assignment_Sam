# 模块说明：`evaluate.py`

## 1. 模块作用
`evaluate.py` 对训练后模型做定量评估，并与随机策略做基准对比。

## 2. 核心功能
- 每关运行固定局数（默认100）
- 统计通关率、平均步数、死锁率
- 计算 Random Agent 通关率作为基线
- 输出评估图与指标文件

## 3. 使用到的知识点
- 离线策略评估
- 基准对照实验（Baseline）
- 指标可视化

## 4. 参数说明（可调性 + 意义）

| 参数 | 是否可调 | 意义 |
|---|---|---|
| `--model` | 可调 | 待评估模型路径 |
| `--episodes` | 可调 | 每关评估局数（越大越稳定） |
| `--results-dir` | 可调 | 评估图和指标保存目录 |

## 5. 关键变量说明

| 变量 | 是否可调 | 意义 |
|---|---|---|
| `successes` | 运行时变量 | 通关次数 |
| `steps_on_success` | 运行时变量 | 成功局步数列表 |
| `deadlocks` | 运行时变量 | 死锁次数 |
| `win_rate` | 指标结果 | 通关率 |
| `avg_steps` | 指标结果 | 成功局平均步数 |
| `deadlock_rate` | 指标结果 | 死锁率 |

## 6. 输出文件
- `results/evaluation_results.png`
- `results/evaluation_metrics.npz`

## 7. 与作业要求对应
- 满足“评估结果与实验分析”要求
- 支持报告中的“与基线对比”章节
