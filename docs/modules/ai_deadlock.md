# 模块说明：`ai/deadlock.py`

## 1. 模块作用
`deadlock.py` 检测 Sokoban 典型无解状态，避免训练把时间浪费在不可恢复轨迹上。

## 2. 核心功能
- 角落死锁 `is_corner_deadlock`
- 边缘死锁 `is_edge_deadlock`
- 冻结死锁 `is_freeze_deadlock`
- 综合入口 `is_deadlock`

## 3. 使用到的知识点
- 组合搜索中的剪枝思想
- 递归依赖检测（冻结死锁）
- RL 训练中的失败状态早停

## 4. 变量说明（可调性 + 意义）

| 变量 | 是否可调 | 意义 |
|---|---|---|
| `walls` | 关卡固定 | 墙体集合 |
| `targets` | 关卡固定 | 目标点集合 |
| `all_boxes` | 运行时变量 | 当前所有箱子坐标集合 |
| `map_width/map_height` | 关卡可调 | 边缘死锁判断边界 |
| `visited`（递归局部） | 不需调参 | 防止冻结检测循环引用 |
| `corners`（局部） | 不需调参 | 四种角落组合逻辑 |

## 5. 与作业要求对应
- 完整覆盖文档中三类死锁检测算法
- 直接参与奖励惩罚（`deadlock = -50`）和终止条件
