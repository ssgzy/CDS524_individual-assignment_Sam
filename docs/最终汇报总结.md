# ShadowBox 作业最终汇报总结（CDS524 Assignment 1）

## 1. 项目目标与完成情况
本项目完成了一个基于 DQN 的多机制推箱子强化学习系统 ShadowBox，满足作业要求中的四大块：
- Game Design：5关递进式地图 + 传送门/压力板/机关门/子关卡
- Q-Learning Implementation：Dueling Double DQN + PER + epsilon-greedy
- Game Interaction：Pygame 伪3D等距可视化，支持手动与AI演示
- Documentation：模块级中文文档、训练评估脚本、汇报结构化总结

## 2. 与评分Rubric对照

### 2.1 Game Design（10分）
- 目标与规则：推所有箱子到同色目标点；超时/死锁失败
- 状态空间：36维向量（层级、玩家、箱子、目标、门、压力板、步数等）
- 动作空间：5动作（上/下/左/右/ENTER）
- 奖励函数：包含正负奖励 + 势能奖励 shaping

### 2.2 Q-Learning Implementation（10分）
- DQN核心算法：`ai/dqn_model.py` + `ai/agent.py`
- 学习策略：epsilon-greedy、gamma、learning rate
- 稳定化机制：Double DQN、目标网络、梯度裁剪
- 采样机制：Prioritized Experience Replay

### 2.3 Game Interaction（10分）
- UI框架：Pygame
- 可视化：`game/renderer.py` 等距渲染 + 实时状态显示
- 交互：`play.py` 手动控制、AI自动演示

### 2.4 Documentation & Presentation（10分）
- 模块文档：`docs/modules/*.md`
- 最终总结：`docs/最终汇报总结.md`
- 训练与评估图表：`results/training_results.png`, `results/evaluation_results.png`

## 3. 关键技术亮点
1. 稀疏奖励缓解：势能奖励（距离塑形）
2. 无解状态处理：角落/边缘/冻结死锁检测
3. 高难机制集成：嵌套子关卡 + 门控机关 + 传送映射
4. 训练稳定性：Dueling + Double + PER + curriculum

## 4. 主要知识点（用于口头汇报）
- Markov Decision Process（状态、动作、奖励、转移）
- Q-Learning 与 Bellman Target
- Deep Q-Network（用神经网络近似Q函数）
- Dueling DQN（V与A分流）
- Double DQN（在线选动作、目标网估值）
- Prioritized Replay（按TD误差采样）
- Reward Shaping（势函数差值）
- Curriculum Learning（由易到难）
- Deadlock Pruning（无解路径剪枝）
- Isometric Rendering（二维网格的伪3D视觉）

## 5. 超参数总览（可调项）

| 参数 | 默认值 | 是否可调 | 作用 |
|---|---:|---|---|
| `state_dim` | 36 | 可调（需联动） | 网络输入维度 |
| `action_dim` | 5 | 可调（需联动） | 网络输出动作数 |
| `learning_rate` | 0.0005 | 可调 | 收敛速度与稳定性 |
| `gamma` | 0.99 | 可调 | 长期奖励权重 |
| `epsilon_start/end/decay` | 1.0/0.05/0.995 | 可调 | 探索利用平衡 |
| `batch_size` | 64 | 可调 | 训练噪声与效率 |
| `buffer_capacity` | 10000 | 可调 | 经验多样性 |
| `target_update_freq` | 500 | 可调 | 训练稳定性 |
| `per_alpha/beta` | 0.6/0.4 | 可调 | PER强度与偏差修正 |
| `grad_clip` | 1.0 | 可调 | 防止梯度爆炸 |
| `curriculum_threshold/window` | 0.8/20 | 可调 | 关卡升级速度 |
| `reward weights` | 文档默认值 | 可调 | 策略学习方向 |

## 6. 文件结构与职责

## 7. 运行与复现实验
```bash
pip install -r requirements.txt
python train.py --episodes 10000
python evaluate.py --model checkpoints/final_model.pth --episodes 100
python play.py --mode agent --model checkpoints/final_model.pth --level 5
```
